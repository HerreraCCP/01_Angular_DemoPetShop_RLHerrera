## Getting Started

This is an instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Prerequisites

.NodeJS - is required to run this project..
* [NodeJs](https://nodejs.org/en/download/)

MySql - I used for the database we are using MySql
* [MySql](https://dev.mysql.com/downloads/installer/)

### Installation

1. Visit the official Node.js page and download the latest.
   ```
   [NodeJs](https://nodejs.org/en/download/)
   ```
2. Install TypeScript 
   ```
   npm install -g typescript
   ```
4. Install Angular CLI
   ```
   npm install -g @angular/cli
   ```  
5. Install MySql.
   ```
   [MySql](https://dev.mysql.com/downloads/installer/)
   ```
6. Clone the project.
    ```
   [Project](https://github.com/HerreraCCP/01_Angular_Demo_RLHerrera_PetShop)
   ```
7. Clone the project.
    ```
   npm install -g
   ```
8. By pass login (if u prefer).
    ```
   If you want to pass the authentication in the file "auth.service.ts" in "line 12", put the "negacao(!)". We are using mockoon too
   ```
9. Execute.
    ```
   ng server -o
   ```
